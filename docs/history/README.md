# Historical documents

These are **superseded planning and progress artifacts** from earlier phases of
Luminframe's development, kept for the record. They describe past states, roadmaps,
and audits that have since been completed or replaced.

**They are not a current map of the codebase.** For how the system is built today, see
[../../ARCHITECTURE.md](../../ARCHITECTURE.md) (the canonical pattern language) and
[../../README.md](../../README.md). Per that pattern language's rule that "comments
describe the present," nothing here should be trusted as describing the code as it now
stands.
